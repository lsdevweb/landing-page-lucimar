# Landing Page - Lucimar Santos
## Descrição

Este é um projeto de **Landing Page profissional** para apresentar serviços e produtos, focado em conversão de visitantes em clientes.  
O layout é limpo, responsivo e pronto para hospedar no **GitHub Pages**, **Netlify**, **InfinityFree** ou qualquer hospedagem que suporte HTML.

---

## Recursos

- Layout responsivo com **Tailwind CSS**
- Ícones do **Font Awesome**
- **Google Tag Manager** integrado (`GTM-NZZQQ2LF`)
- Seções:  
  - Hero / Apresentação  
  - Benefícios / Serviços  
  - Chamada para ação (WhatsApp)  
  - Rodapé com informações e versículo
- Campos de **telefone e e-mail** comentados para fácil personalização
- Efeitos de hover nos cards e botões
